## 1. Overview

**Problem**: Web pages contain noise (ads, scripts, navigation) that interferes with accurate summarization and metadata extraction.

**Solution**: Content extraction service that fetches URLs, parses HTML, and returns clean text content ready for AI processing.

**Goals**: High-quality text extraction, noise removal, SSRF protection, graceful error handling

**Non-Goals**: JavaScript rendering (use static HTML only), video/audio transcription, PDF parsing

## 3. User Story / Behavior

As a Briefly user sharing a news article, I want the system to extract only the article text, so that the AI summary is accurate and not confused by ads or navigation.

### Key Behaviors

- User shares a URL via share sheet
- System fetches HTML content from URL
- Removes non-content elements (scripts, styles, nav, footer, header, aside, form)
- Identifies main content container (article, main, or body)
- Extracts and cleans text
- Returns clean text for summarization

## 6. Implementation

### Files

- `src/ingestion/extractor.py` - ContentExtractor service with SSRF protection
- `src/ingestion/exceptions.py` - ExtractionError, SSRFError
- `src/utils/http_client.py` - Async HTTP client pool

### Key Logic

1. **URL Validation & SSRF Protection**:
   - Validate URL format (http:// or https://)
   - Resolve DNS and check IP address
   - Block private IPs (10.x, 172.16-31.x, 192.168.x)
   - Block localhost and link-local addresses

2. **Content Extraction**:
   - Fetch HTML using async HTTP client (httpx)
   - Parse HTML with BeautifulSoup
   - Remove noise tags: script, style, nav, footer, header, aside, form
   - Find main content: article, main, or body
   - Extract text and normalize whitespace

3. **Error Handling**:
   - 404/500 errors → ExtractionError
   - Timeout → ExtractionError
   - No text found → ExtractionError
   - SSRF detection → SSRFError

## 7. Edge Cases

| Scenario | Handling |
|----------|----------|
| Invalid URL format | Raise ExtractionError |
| 404/500 response | Raise ExtractionError |
| DNS resolution fails | Raise ExtractionError |
| Private IP detected | Raise SSRFError |
| JavaScript-heavy page (empty HTML) | Raise ExtractionError (no text found) |
| Timeout after 10s | Raise ExtractionError |
| Redirect (301/302) | Follow redirect, extract from final URL |
