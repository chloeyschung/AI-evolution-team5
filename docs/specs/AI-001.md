# AI-001: Core 3-Line Summarizer

## User Story
As a Briefly user, I want to receive a concise 3-line summary of any content I ingest, so that I can quickly understand the core message without reading the full text.

## Dependencies
- **depends_on:** none
- **depended_by:** `@specs/UX-001.md`, `@specs/UX-003.md`
- **dependency code:** none

## Implementation Contract

### Level 1: Natural Language
The `Summarizer` service takes raw text (or extracted content from a URL) as input and uses a cloud-based LLM (e.g., Anthropic Claude) to generate exactly three concise, high-density lines of text. The output must capture the "Who, What, and Why" of the content.

### Level 2: Logic Flow (Pseudocode)
```
INPUT: raw_content (string), max_lines (int, default=3)
PROCESS:
  1. Clean input text (remove excessive whitespace, boilerplate).
  2. Construct prompt for LLM:
     - Role: Expert Summarizer.
     - Task: Summarize the following content into exactly {max_lines} lines.
     - Constraint: High density, no fluff, preserve context.
  3. Call Cloud LLM API with prompt and raw_content.
  4. Parse response to ensure it adheres to the line count constraint.
  5. Handle API errors (retry logic or fallback to a simpler model).
OUTPUT: summary_text (string)
ERROR: SummarizationError (if API fails or content is unprocessable)
```

### Level 3: Formal Interface
```python
class Summarizer:
    def summarize(self, content: str, max_lines: int = 3) -> str:
        """
        Generates a concise summary of the provided content.
        
        Args:
            content: The raw text to be summarized.
            max_lines: The maximum number of lines for the summary.
            
        Returns:
            A string containing the summarized text.
            
        Raises:
            SummarizationError: If the service fails to generate a valid summary.
        """
        pass
```

## Verification Contract

### Level 1: Scenarios (Natural Language)
- **Success:** A long news article is condensed into three meaningful, distinct lines.
- **Failure:** The LLM API is unreachable; the service raises a `SummarizationError`.
- **Edge:** The input content is extremely short (e.g., a single sentence); the service returns the sentence or a slightly expanded version without exceeding 3 lines.
- **Edge:** The input content is empty or nonsimulated; the service raises an error.

### Level 2: Given-When-Then
```
GIVEN a long article about the history of space exploration
WHEN the summarize method is called
THEN the output is a string with exactly 3 lines of high-density text

GIVEN the LLM API returns a 500 error
WHEN the summarize method is called
THEN a SummarizationError is raised
```
