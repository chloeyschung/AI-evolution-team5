## 1. Overview

### 1.1. Problem Statement

Users save content but don't consistently consume it. Without reminders, saved content accumulates unread, defeating the purpose of the knowledge management system. Smart reminders can nudge users to maintain their consumption habits without being annoying.

### 1.2. Solution

A smart reminder system that:
- Tracks user's consumption patterns (preferred times, frequency)
- Sends timely reminders based on user preferences and behavior
- Avoids spamming by learning from user responses
- Provides contextual reminders (e.g., "You have 5 unread items from your favorite platform")

### 1.3. Goals

- Increase daily content consumption by 25%
- Reduce content backlog accumulation
- Provide personalized, non-intrusive reminders
- Learn from user behavior to optimize reminder timing

### 1.4. Non-Goals

- External push notifications (requires mobile app integration - Phase 3)
- Email reminders (Phase 2)
- Social reminders ("Your friend read this") - Phase 4

## 3. Reminder Types

### 3.1. Backlog Reminder

Triggered when user has unread content exceeding threshold.

```
"You have 15 unread items in your inbox. Ready to catch up?"
```

### 3.2. Streak Reminder

Triggered when user's streak is at risk (no activity today).

```
"Don't break your 7-day streak! You have 8 items waiting."
```

### 3.3. Time-Based Reminder

Triggered at user's preferred consumption time.

```
"Time for your daily knowledge boost! 5 new items since yesterday."
```

### 3.4. Re-engagement Reminder

Triggered after user hasn't used the app for N days.

```
"We miss you! 23 items are waiting for you in Briefly."
```

### 4.1. New Models

#### ReminderPreference

```python
class ReminderPreference(Base):
    """User's reminder preferences."""

    __tablename__ = "reminder_preferences"

    id = Column(Integer, primary_key=True)
    user_id = Column(Integer, ForeignKey("user_profile.id"), unique=True, nullable=False)
    is_enabled = Column(Integer, default=1)  # SQLite boolean
    preferred_time = Column(Time)  # Preferred reminder time (e.g., 18:00)
    frequency = Column(String(20), default="daily")  # 'daily', 'weekly', 'never'
    quiet_hours_start = Column(Time)  # Don't remind before this time
    quiet_hours_end = Column(Time)  # Don't remind after this time
    backlog_threshold = Column(Integer, default=10)  # Items before backlog reminder
    updated_at = Column(DateTime, default=utc_now, onupdate=utc_now)
```

class ReminderLog(Base):

## 6. Implementation Details

### 6.1. ReminderEngine Class

```python
class ReminderEngine:
    """Generate and manage smart reminders."""

    def __init__(self, db_session: AsyncSession):
        self._preference_repo = ReminderPreferenceRepository(db_session)
        self._pattern_repo = UserActivityPatternRepository(db_session)
        self._log_repo = ReminderLogRepository(db_session)

    async def get_suggestion(self, user_id: int) -> Optional[ReminderSuggestion]:
        """Get current reminder suggestion for user.

        Returns:
            ReminderSuggestion or None if no reminder needed
        """
        # Check if reminders enabled
        preferences = await self._preference_repo.get(user_id)
        if not preferences or not preferences.is_enabled:
            return None

        # Check quiet hours
        if self._is_quiet_hours(preferences):
            return None

        # Check frequency limits
        if not await self._can_send_reminder(user_id, preferences):
            return None

        # Generate reminder based on conditions
        return await self._generate_reminder(user_id, preferences)

    async def _generate_reminder(
        self, user_id: int, preferences: ReminderPreference
    ) -> Optional[ReminderSuggestion]:
        """Generate appropriate reminder based on user state."""
        # Priority order: streak > backlog > time_based > reengagement
        streak_reminder = await self._check_streak_reminder(user_id)
        if streak_reminder:
            return streak_reminder

        backlog_reminder = await self._check_backlog_reminder(
            user_id, preferences
        )
        if backlog_reminder:
            return backlog_reminder

        # ... other reminder types
```

### 6.2. Activity Pattern Learning

```python
class ActivityPatternLearner:
    """Learn user activity patterns from swipe history."""

    async def update_patterns(self, user_id: int) -> UserActivityPattern:
        """Update user's activity patterns based on recent history."""
        # Get last 30 days of swipe history
        swipes = await self._get_recent_swipes(user_id, days=30)

        if not swipes:
            return await self._get_or_create_pattern(user_id)

        # Calculate most active hour
        hour_counts = self._count_by_hour(swipes)
        most_active_hour = max(hour_counts.keys(), key=lambda h: hour_counts[h])

        # Calculate most active day
        day_counts = self._count_by_day(swipes)
        most_active_day = max(day_counts.keys(), key=lambda d: day_counts[d])

        # Update pattern
        pattern = await self._get_or_create_pattern(user_id)
        pattern.most_active_hour = most_active_hour
        pattern.most_active_day = most_active_day
        pattern.avg_daily_swipes = len(swipes) / 30

        return pattern
```

## 7. Edge Cases

| Scenario | Handling |
|----------|----------|
| User has no swipe history | Use default preferences (6 PM daily) |
| User disabled reminders | Return None from get_suggestion |
| User in quiet hours | Suppress all reminders |
| User just received reminder | Respect frequency limit (max 1/day by default) |
| No unread content | Don't send backlog reminder |
