# ADV-002: Gamified Achievement System

**Status**: Spec | **Created**: 2026-04-14 | **Author**: abraxaspark

---

## 1. Overview

**Problem**: Users lack motivation to consistently engage with saved content. No visual feedback for building habits or milestones.

**Solution**: Achievement system rewarding streaks, swipe milestones, platform diversity, and curation.

**Goals**: +20% DAU, +15% swipe completion, visual progress feedback

**Non-Goals**: Social leaderboards, competitive elements, external rewards (Phase 4+)

## 4. Data Models

### New Tables

**achievement_definitions**: id, key, type, name, description, icon, trigger_value, is_active

**user_achievements**: id, user_id, achievement_id, unlocked_at, metadata_json
- Unique constraint: (user_id, achievement_id)

**user_streaks**: id, user_id, current_streak, longest_streak, last_activity_date, total_active_days, updated_at
- Unique constraint: user_id

### Existing Used
- `SwipeHistory` (volume stats)
- `Content` (platform diversity, kept count)
- `UserProfile` (link to users)

---

## 5. API

| Endpoint | Description |
|----------|-------------|
| `GET /achievements` | All definitions with progress (optional `?type=streak`) |
| `GET /achievements/stats` | Summary: total_unlocked, completion%, streak |
| `POST /achievements/check` | Check & award new (called after swipe) |

**Response example**:
```json
{ achievements: [{ id, key, type, name, icon, trigger_value, is_unlocked, progress, progress_percent, unlocked_at }] }
```

## 6. Implementation

**Files**:
- `src/ai/achievement_checker.py` - Check & award logic
- `src/data/achievement_repository.py` - CRUD
- `src/data/streak_repository.py` - Streak tracking
- `src/data/seed_achievements.py` - Initialize 16 achievements
- `src/api/schemas.py` - Response models
- `src/api/routes.py` - Endpoints

**Streak Logic**: Count consecutive days ending at last_activity_date. Gap > 1 day breaks streak.

---

## 7. Edge Cases

| Scenario | Handling |
|----------|----------|
| No swipe history | Empty achievements, 0 progress |
| Multiple unlocks at once | Award all in transaction |
| Delete all content | Streak resets, volume remains (historical) |
| Definition disabled | Don't award, keep existing |
