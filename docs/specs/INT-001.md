# INT-001: YouTube Auto-Sync

**Feature ID**: F-022 (YouTube Integration)
**Status**: Spec
**Priority**: Phase 2 - Integrations

## Overview

Automatically ingest YouTube "Watch Later" videos or specific playlists via the YouTube Data API v3. Users can connect their YouTube account and enable automatic syncing of selected playlists.

## Requirements

### Functional Requirements

1. **YouTube OAuth Connection**
   - Connect YouTube account via OAuth 2.0
   - Store OAuth tokens securely
   - Handle token refresh automatically
   - Disconnect option

2. **Playlist Selection**
   - List user's playlists
   - Select "Watch Later" (default playlist)
   - Select custom playlists
   - Multi-select support

3. **Sync Configuration**
   - Sync frequency (hourly, daily, weekly)
   - Auto-start sync on connection
   - Manual sync trigger
   - Sync status display

4. **Content Ingestion**
   - Fetch new videos from selected playlists
   - Skip already-ingested videos (by video ID)
   - Extract video metadata (title, channel, thumbnail, publish date)
   - Generate AI summary for each video

5. **Conflict Handling**
   - Detect duplicate URLs
   - Skip if already exists in database
   - Update metadata if video info changed

### Non-Functional Requirements

1. **Performance**
   - Sync 100 videos in < 30 seconds
   - Non-blocking background sync
   - Rate limit awareness (YouTube API quotas)

2. **Reliability**
   - Retry failed syncs (exponential backoff)
   - Queue-based processing
   - Error logging and notification

3. **Security**
   - OAuth tokens encrypted at rest
   - Minimal scope request (read-only)
   - Token rotation on expiry

### Key Components

#### 1. YouTube API Client

```python
# src/integrations/youtube/client.py
from googleapiclient.discovery import build
from google_auth_oauthlib.flow import InstalledAppFlow

class YouTubeClient:
    def __init__(self, credentials: Credentials):
        self.service = build('youtube', 'v3', credentials=credentials)
        self.user_id: str | None = None

    async def get_user_id(self) -> str:
        """Get authenticated user's channel ID."""
        response = await self.service.channels().list(
            part='id',
            mine=True
        ).execute()
        self.user_id = response['items'][0]['id']
        return self.user_id

    async def get_playlists(self) -> list[Playlist]:
        """Get user's playlists."""
        response = await self.service.playlists().list(
            part='id,snippet',
            mine=True,
            maxResults=50
        ).execute()
        return [self._parse_playlist(p) for p in response.get('items', [])]

    async def get_playlist_items(self, playlist_id: str) -> list[Video]:
        """Get videos in a playlist."""
        videos = []
        page_token = None

        while True:
            response = await self.service.playlistItems().list(
                part='snippet,contentDetails',
                playlistId=playlist_id,
                maxResults=50,
                pageToken=page_token
            ).execute()

            videos.extend([self._parse_video_item(p) for p in response.get('items', [])])

            if 'nextPageToken' not in response:
                break
            page_token = response['nextPageToken']

        return videos
```

#### 2. Sync Service

```python
# src/integrations/youtube/sync.py
class YouTubeSyncService:
    def __init__(
        self,
        youtube_client: YouTubeClient,
        content_repo: ContentRepository,
        integration_repo: IntegrationRepository
    ):
        self.youtube = youtube_client
        self.content_repo = content_repo
        self.integration_repo = integration_repo

    async def sync_playlist(
        self,
        user_id: int,
        playlist_id: str,
        since: datetime | None = None
    ) -> SyncResult:
        """Sync a playlist, ingesting new videos."""
        # Get playlist items from YouTube
        videos = await self.youtube.get_playlist_items(playlist_id)

        # Get last sync time
        last_sync = await self.integration_repo.get_last_sync(
            user_id, 'youtube', playlist_id
        )

        # Filter new videos
        new_videos = [
            v for v in videos
            if v.published_at and v.published_at > (last_sync or datetime.min)
        ]

        # Ingest each video
        ingested = 0
        skipped = 0
        errors = []

        for video in new_videos:
            try:
                url = f"https://youtube.com/watch?v={video.video_id}"

                # Check if already exists
                existing = await self.content_repo.get_by_url(url)
                if existing:
                    skipped += 1
                    continue

                # Create content
                content = await self.content_repo.save(
                    platform='youtube',
                    content_type='video',
                    url=url,
                    title=video.title,
                    author=video.channel_title,
                    timestamp=video.published_at,
                    thumbnail_url=video.thumbnail_url,
                )

                # Generate summary
                await self.summarizer.summarize(content)

                ingested += 1

            except Exception as e:
                errors.append({'video_id': video.video_id, 'error': str(e)})

        # Update sync timestamp
        await self.integration_repo.update_last_sync(
            user_id, 'youtube', playlist_id, datetime.now(timezone.utc)
        )

        return SyncResult(ingested=ingested, skipped=skipped, errors=errors)
```

# src/data/models.py

class IntegrationTokens(Base):
    """OAuth tokens for third-party integrations."""

    __tablename__ = "integration_tokens"

    id = Column(Integer, primary_key=True, index=True)
    user_id = Column(Integer, ForeignKey("user_profile.id"), nullable=False, index=True)
    provider = Column(String(50), nullable=False, index=True)  # 'youtube', 'linkedin'
    access_token = Column(Text, nullable=False)
    refresh_token = Column(Text, nullable=False)
    expires_at = Column(DateTime, nullable=False, index=True)
    created_at = Column(DateTime, default=utc_now, nullable=False)
    updated_at = Column(DateTime, default=utc_now, onupdate=utc_now, index=True)

    __table_args__ = (
        sqlalchemy.UniqueConstraint("user_id", "provider", name="unique_user_provider"),
    )


class IntegrationSyncConfig(Base):
    """Sync configuration for integrations."""

    __tablename__ = "integration_sync_configs"

    id = Column(Integer, primary_key=True, index=True)
    user_id = Column(Integer, ForeignKey("user_profile.id"), nullable=False, index=True)
    provider = Column(String(50), nullable=False)
    resource_id = Column(String(100), nullable=False)  # Playlist ID, etc.
    resource_name = Column(String(200))  # Playlist name
    sync_frequency = Column(String(20), nullable=False)  # 'hourly', 'daily', 'weekly'

    is_active = Column(Integer, nullable=False, default=1)
    created_at = Column(DateTime, default=utc_now, nullable=False)
    updated_at = Column(DateTime, default=utc_now, onupdate=utc_now, index=True)


class IntegrationSyncLog(Base):
    """Log of sync operations."""

    __tablename__ = "integration_sync_logs"

    id = Column(Integer, primary_key=True, index=True)
    user_id = Column(Integer, ForeignKey("user_profile.id"), nullable=False, index=True)
    provider = Column(String(50), nullable=False)

2. **API Quotas**: Monitor and respect YouTube API rate limits (1000 units/day)
3. **Scope**: Request minimal scope (`https://www.googleapis.com/auth/youtube.readonly`)
4. **Token Storage**: Use secure database column with encryption
