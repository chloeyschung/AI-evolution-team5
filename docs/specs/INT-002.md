# INT-002: LinkedIn/Social Sync

**Feature ID**: F-023 (LinkedIn/Social Integration)
**Status**: Spec
**Priority**: Phase 2 - Integrations

## Overview

Automatically ingest LinkedIn saved posts and other social media content via platform APIs. Users can connect their social accounts and enable automatic syncing of saved content.

## Requirements

### Functional Requirements

1. **LinkedIn OAuth Connection**
   - Connect LinkedIn account via OAuth 2.0
   - Store OAuth tokens securely
   - Handle token refresh automatically
   - Disconnect option

2. **Saved Posts Sync**
   - Fetch user's saved posts from LinkedIn
   - Extract post metadata (title, author, timestamp, URL)
   - Skip already-ingested posts (by URL)
   - Handle different post types (article, text, video, image)

3. **Sync Configuration**
   - Sync frequency (hourly, daily, weekly)
   - Auto-start sync on connection
   - Manual sync trigger
   - Sync status display

4. **Content Ingestion**
   - Fetch saved posts from LinkedIn API
   - Extract post content and metadata
   - Generate AI summary for each post
   - Handle rate limits gracefully

5. **Conflict Handling**
   - Detect duplicate URLs
   - Skip if already exists in database
   - Update metadata if post info changed

### Non-Functional Requirements

1. **Performance**
   - Sync 50 posts in < 30 seconds
   - Non-blocking background sync
   - Rate limit awareness (LinkedIn API quotas)

2. **Reliability**
   - Retry failed syncs (exponential backoff)
   - Queue-based processing
   - Error logging and notification

3. **Security**
   - OAuth tokens encrypted at rest
   - Minimal scope request (read-only)
   - Token rotation on expiry

## Security Considerations

1. **OAuth Tokens**: Encrypt at rest using AES-256
2. **API Quotas**: Monitor and respect LinkedIn API rate limits
3. **Scope**: Request minimal scope (read-only access to saved items)
4. **Token Storage**: Use existing `IntegrationTokens` model with encryption

2. **Integration Tests**: Use LinkedIn API test credentials
3. **E2E Tests**: Manual testing with real LinkedIn account
