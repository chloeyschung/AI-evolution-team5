## 1. Overview

**Problem**: Users need to organize and filter their saved content by source platform, content type, and publication date.

**Solution**: Automated metadata extraction from URLs and HTML content, identifying platform, content type, timestamp, author, and title.

**Goals**: Accurate platform detection, comprehensive metadata extraction, graceful fallback for unknown sources

**Non-Goals**: Manual metadata editing, metadata validation/correction by users

## 3. User Story / Behavior

As a Briefly user, I want my saved content to automatically show the source platform and content type, so that I can filter and organize my knowledge library effectively.

### Key Behaviors

- System parses URL to extract domain and path
- Matches domain against known platforms (YouTube, LinkedIn, Medium, etc.)
- Determines content type from URL path patterns
- Extracts timestamp, author, title from HTML meta tags
- Stores metadata with content for filtering and display

## 6. Implementation

### Files

- `src/ai/metadata_extractor.py` - MetadataExtractor service
- `src/ai/exceptions.py` - MetadataExtractionError
- `src/ai/categorizer.py` - Related: AI tag generation

### Key Logic

1. **Platform Detection** (URL-based):
   - youtube.com, youtu.be → "YouTube"
   - linkedin.com → "LinkedIn"
   - medium.com → "Medium"
   - twitter.com, x.com → "Twitter"
   - instagram.com → "Instagram"
   - Default → "Web"

2. **Content Type Detection** (URL path patterns):
   - /watch, /shorts, /live → "video"
   - /p/, /posts/, /articles/ → "social_post"
   - /pictures/, /reel/, /photo/ → "image"
   - Default → "article"

3. **HTML Metadata Extraction** (if HTML available):
   - Timestamp: `<time datetime>`, og:article:published_time, article:published_time
   - Author: article:author, og:author, `<byline>`
   - Title: og:title, `<title>`
   - Thumbnail: og:image, twitter:image

4. **Fallback Behavior**:
   - Without HTML: extract from URL only
   - Unknown platform: "Web"
   - Unknown type: "article"

## 7. Edge Cases

| Scenario | Handling |
|----------|----------|
| Unknown domain | platform="Web", content_type="article" |
| No HTML content | Extract from URL only |
| Short URL (youtu.be) | Detect as YouTube |
| Multiple timestamps in HTML | Use first/most specific |
| No author found | author=None |
