## Key Design Decisions (recorded from plan)
- R2 config optional at startup, validated at use-time in R2Client
- ScreenshotImage.content_id uses regular FK; Content.screenshot_image_id uses use_alter=True to break circular FK
- ImageProcessor accepts optional summarizer + r2_client (backwards-compat: no-arg constructor still works for ShareHandler)
- delete order in conftest: TrustedSource → ScreenshotImage → Content → UserProfile
- domain normalization: host[4:] if host.startswith("www.") else host (prefix-strip, not .replace)
